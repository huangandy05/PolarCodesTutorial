# Polar Codes Tutorial

This repository contains a Markdown tutorial wiki for learning **polar codes** from first principles. It covers the intuition, mathematical foundations, encoding, frozen-bit selection, SC and SCL decoding, implementation guidance, 5G context, research directions, exercises, glossary terms, and references.

The tutorial is designed as a progressive learning resource for readers with basic background in probability, linear algebra, binary arithmetic, and digital communications.

## Repository Structure

- `index.md`: main tutorial overview and learning path.
- `01-intuition.md` to `12-references.md`: linked tutorial chapters.
- `AGENTS.md`: authoring instructions used to generate and maintain the tutorial.
